# Jarvis 订阅配置故障排查手册

## 概述

本手册提供了 Jarvis 订阅配置相关的常见问题和解决方案。

## 常见问题分类

### 1. 配置文件生成问题

#### 问题：生成配置文件失败
**症状**：
- 支付成功后无法生成配置文件
- 显示"生成配置文件失败，请联系客服"

**可能原因**：
- 订单状态异常
- 服务器内部错误
- 数据库连接问题

**解决方案**：
1. 检查订单是否已支付成功
2. 查看浏览器控制台错误日志
3. 联系客服提供订单号

#### 问题：缺少订单ID
**症状**：
- 显示"缺少订单ID"错误

**可能原因**：
- 前端请求参数不完整
- 订单创建失败

**解决方案**：
1. 重新进行购买流程
2. 检查网络连接
3. 清除浏览器缓存后重试

### 2. 配置文件下载问题

#### 问题：下载配置文件失败
**症状**：
- 点击下载链接无反应
- 显示"配置不存在或无权限"

**可能原因**：
- 配置文件已过期
- 用户权限不足
- 网络连接问题

**解决方案**：
1. 检查配置文件是否已过期（7天有效期）
2. 重新登录账户
3. 联系客服重新生成配置

#### 问题：配置文件已过期
**症状**：
- 显示"配置已过期"错误
- 无法下载配置文件

**可能原因**：
- 配置文件超过7天有效期
- 时间戳验证失败

**解决方案**：
1. 重新购买订阅服务
2. 联系客服延长配置有效期

### 3. 配置文件解密问题

#### 问题：设备指纹不匹配
**症状**：
- PC端显示"设备指纹不匹配"
- 无法解密配置文件

**可能原因**：
- 配置文件与当前设备不匹配
- 硬件信息发生变化

**解决方案**：
1. 确保在同一台设备上使用
2. 重新下载配置文件
3. 检查系统时间是否正确

#### 问题：数字签名验证失败
**症状**：
- 显示"数字签名验证失败"
- 配置文件解密失败

**可能原因**：
- 配置文件被篡改
- 密钥不匹配

**解决方案**：
1. 重新下载配置文件
2. 检查网络传输是否完整
3. 联系客服重新生成配置

#### 问题：配置文件已过期（时间戳）
**症状**：
- 显示"配置文件已过期"
- 时间戳验证失败

**可能原因**：
- 配置文件生成时间超过5分钟
- 系统时间不正确

**解决方案**：
1. 重新生成配置文件
2. 校准系统时间
3. 检查网络延迟

#### 问题：解密失败
**症状**：
- 显示"解密失败"
- 自动重试3次后仍然失败

**可能原因**：
- 配置文件损坏
- 加密密钥错误
- 算法不匹配

**解决方案**：
1. 重新下载配置文件
2. 检查PC端服务版本
3. 联系技术支持

### 4. API密钥验证问题

#### 问题：API密钥无效或已过期
**症状**：
- 显示"API密钥无效或已过期"
- 无法使用AI服务

**可能原因**：
- API密钥已过期
- API密钥被吊销
- 设备指纹不匹配

**解决方案**：
1. 检查API密钥有效期
2. 重新生成配置文件
3. 确认设备指纹匹配

#### 问题：API密钥使用频率超限
**症状**：
- 显示"API密钥使用频率超限"
- 服务被暂时封禁

**可能原因**：
- 每小时调用超过100次
- 每天调用超过1000次

**解决方案**：
1. 等待封禁期结束
2. 联系客服申请解封
3. 升级到更高级别的套餐

### 5. 网络连接问题

#### 问题：网络连接超时
**症状**：
- 请求超时
- 无法连接到服务器

**可能原因**：
- 网络不稳定
- 防火墙阻止
- DNS解析失败

**解决方案**：
1. 检查网络连接
2. 尝试使用其他网络
3. 配置代理服务器

#### 问题：HTTPS证书错误
**症状**：
- 显示证书错误
- 无法建立安全连接

**可能原因**：
- 系统时间不正确
- 证书已过期
- 中间人攻击

**解决方案**：
1. 校准系统时间
2. 更新根证书
3. 检查网络安全性

### 6. 浏览器兼容性问题

#### 问题：单选框无法选择
**症状**：
- 模型选择单选框无法点击
- 界面显示异常

**可能原因**：
- 浏览器版本过旧
- JavaScript被禁用
- CSS样式冲突

**解决方案**：
1. 更新浏览器到最新版本
2. 启用JavaScript
3. 清除浏览器缓存
4. 尝试使用其他浏览器

#### 问题：响应式布局异常
**症状**：
- 在小屏幕设备上显示异常
- 界面元素重叠

**可能原因**：
- 浏览器不支持CSS Grid
- 视口设置不正确

**解决方案**：
1. 更新浏览器
2. 检查视口meta标签
3. 使用现代浏览器

## 调试工具

### 1. 浏览器开发者工具

#### 查看控制台日志
```javascript
// 在浏览器控制台执行
console.log('当前用户信息:', localStorage.getItem('token'));
console.log('当前页面URL:', window.location.href);
```

#### 检查网络请求
1. 打开开发者工具 (F12)
2. 切换到 Network 标签
3. 重现问题
4. 查看失败的请求详情

### 2. 系统信息收集

#### 收集设备信息
```javascript
const deviceInfo = {
  userAgent: navigator.userAgent,
  screenResolution: `${window.screen.width}x${window.screen.height}`,
  timezone: Intl.DateTimeFormat().resolvedOptions().timeZone,
  language: navigator.language,
  platform: navigator.platform,
  cookieEnabled: navigator.cookieEnabled,
  onLine: navigator.onLine
};
console.log('设备信息:', JSON.stringify(deviceInfo, null, 2));
```

#### 检查本地存储
```javascript
console.log('本地存储内容:');
for (let i = 0; i < localStorage.length; i++) {
  const key = localStorage.key(i);
  const value = localStorage.getItem(key);
  console.log(`${key}: ${value}`);
}
```

### 3. PC端服务诊断

#### 检查配置文件
```powershell
# Windows PowerShell
Get-Content "C:\ProgramData\ModelService\config.json" | ConvertFrom-Json
```

#### 检查服务状态
```powershell
# 检查服务是否运行
Get-Service -Name "JarvisConfigService"

# 查看服务日志
Get-EventLog -LogName Application -Source "JarvisConfigService" -Newest 10
```
